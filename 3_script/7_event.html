BOM[브라우저 오브젝트 모델] DOM [도큐먼트 오브젝트 모델]
onchange : html요소가 변경됨
onclick : 사용자가 html요소를 클릭할때
onmouseover : 사용자가 html요소위로 마우스를 이동
onmouseout : 사용자가 html요소에서 마우스를 멀리 이동
onkeydown: : 사용자가 키보드 키를 누를때
onload: 브라우저가 페이지 로딩을 완료할때
