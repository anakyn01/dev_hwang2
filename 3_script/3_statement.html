자바스크립트 프로그램

컴퓨터 프로그램은 컴퓨터가 실행해야 할 지시 목록입니다
이러한 프로그래밍 명령어를 명령문 이라고 합니다
대부분에 자바스크립트 프로그램에은 많은 문장이 포함되어 있습니다
자바스크립트 문은 아래와 같이 구성됩니다
- 값,연산자,표현식,키워드,주석
- 자바스크립트 예약어 또는 키워드
var : 자바스크립트에서 최초의 변수선언 
변하는 수여서 변수 많은수 여서 변수 
var nums = 3; var nums = 5; 단점이 재선언 재할당을 시켜서
프로그래밍에 혼란을 초래

let : 재선언 재할당을 금지하는 변수 타입 임의대로 만드는 변수들
책상같이 변화가 있거나 조정을 할수 있는것을 사용

const : 재선언 재할당을 금지하는 변수 타입 변화를 할수없고 참조만 할수 있는것들
3.14 cm, mm, r

if : 조건에 따라 실행될 명령문 블록을 표시

switch : 다양한 경우에 실행될 명령문 블록을 표시

for : 루프에서 실행될 명령문 블록을 표시 반복[지정한 범위만큼 실행하는걸 반복한다고 합니다](0~9)
function : 함수를 선언
return : 함수를 종료합니다
try : 문장 블록에 대한 오류처리를 합니다

자바스크립트 키워드는 예약어 입니다 
예약어는 변수 이름으로 사용할수 없습니다
