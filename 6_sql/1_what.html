sql 은 ? 

        - 데이터베이스에 저장, 조작, 검색
-종류 :  mysql, sql server, ms access, oracle, sybase, 
informix, Postgress  

- Data Definition Language : 
create[데이터 베이스 객체 생성],
drop[객체 삭제] 
alter[객체 수정]
Truncate: 테이블내에 모든 데이터는 삭제하지만 
테이블은 남아 있음

- Data Manipulation Language :
select, insert, update, delete

- data control language : 권한부여 제어
grant(특정권한 부여), revoke(회수) 
보안과 접근제어

트랜잭션 : 데이터 베이스에서 하나의 작업단위를 의미
Atomicity(원자성) : 모든 작업은 하나의 단위로 처리됩니다

Consistency(일관성) : 일관성 있는 상태로 유지

Isolation(고립성) : 트랜잭션의 간섭없이 독립적으로 실행

Durability(지속성) : 성공적으로 커밋되면 그 변경사항은 
영구적으로 유지됩니다
속성이라고 부릅니다
